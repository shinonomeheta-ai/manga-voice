# プロジェクトナレッジ（引き継ぎ用コンテキスト）

> Claude Code の制作フローに「これまでの文脈と確定設定」を渡すためのファイル。
> 新しいセッション・協力者・将来の自分が、ゼロから把握できるように書いてある。
> `CLAUDE.md` からも参照される。

---

## 0. このプロジェクトの位置づけ

- 目的: 漫画『ななしちゃん@がんばらない』のシナリオ（ネーム指示書）を量産する。
- 方針: 「単発の賢さ」ではなく「仕組みと蓄積」で量産品質を担保する。
  Claude Code もチャット版も中身のモデルは同じで、1本の文章精度は変わらない。
  価値は自動化・量産・仕組み化（テンプレ固定／設定の自動読み込み／バッチ生成）。
- 形式の判断: カスタムコマンドは推奨形式 `.claude/skills/<名前>/SKILL.md` を採用
  （`/名前` で起動でき、Claude が自動でも使える）。`.claude/commands/` は旧式。

## 1. 作品の骨格

- タイトル: **『ななしちゃん@がんばらない』**
- ジャンル: パチンコ・パチスロ雑学漫画 / 業界裏話
- テーマ: 業界で実際に起きた事件・不正を、フィクションを通じて読者に届ける
- トーン: コメディ＋シリアスのバランス型
- 構成: 1話完結型（事件1つ＝1〜数話）
- 狙い: 打たない読者でも楽しめる雑学エンタメ。史実とフィクションの線引きを明確に。
  業界批判一辺倒にせず、業界の面白さも伝える。

## 3. 登場人物（要点。詳細は 設定/キャラクター設定.md）

- **ななし**（主人公）: 大学2年20歳。読者視点・ツッコミ・聞き手。打たない。
  桐原の事務所でバイト中。あかりの幼馴染で世話役。
- **あかり**: 大学2年20歳。物語の起点・コメディ担当。思い込みが激しく、自分で打つ。
- **桐原**: 30代。桐原調査事務所の所長（探偵）で元業界紙記者。解説役・知識の供給源。
- **九重（ここのえ）**: 掴みどころのないトリックスター悪役。フィクサー/コンサル/
  流れ者（案）。飄々として慇懃。
- **九頭竜（くずりゅう）**: 保身で不正に走る小悪党。ホール店長案が有力。
  気弱で「同情できなくもないが擁護はできない」立ち位置。

相関図: 〈日常パート: ななし―あかり―桐原調査事務所〉と
〈事件パート: 九重→九頭竜→善良な店長/被害者〉の二層構造。

---

## 4. 標準回フォーマット（7パート）

① ツカミ（あかりが何か言い出す）→ ② 疑問の発生（ななし）→
③ 相談（ななしが事務所で桐原に持っていく）→ ④ 解説（桐原が元記者の経験で実在事件を紹介）
→ ⑤ 事件パート（九重・九頭竜らの再現ドラマ）→ ⑥ 現代に戻る（ななしの感想）→
⑦ オチ（あかりの一言 or 後日談）。

回タイプのバリエーション:
- シリアス回: ⑦を軽くせず余韻で締める
- コメディ回: ④⑤を短め、日常パートを厚く
- 連作回: 大型事件を2〜3話に分割
- 依頼パターン回: 関係者や被害者が事務所に依頼に来て、桐原が現在進行形で動く
- 桐原回想回: 元記者時代の事件を桐原視点で語る（縦軸エピソード）

---

## 5. 取り扱い上の配慮（必ず守る）

- 実在事件は桐原の解説として扱い、人物・店舗・メーカー名は架空に置換。
- 不正の手口を詳細に解説（教則化）しない。
- 特定の実在人物・店舗・メーカーの名誉毀損を避ける。反社絡みは慎重に。
- 業界を一方的に悪と描かない。善良な業界人も出す。プレイヤーを馬鹿にしない。
- ギャンブル依存症などはシリアスに扱う。

---

## 6. 縦軸（連載を貫く軸の有力案）

桐原が記者時代に追っていた事件の中心人物が九重で、結果的に書ききれず業界を去った。
これを据えると、シリーズの最終目標まで一本筋が通る。→ 採否は未確定。
