# 第01話「フィーバーの残響」

## ログライン
かつて熱狂で沸いた一軒のホールの栄枯を、元店長の老人の回想でたどり、
「あの熱は、誰のものだったのか」を静かに問う。

## 今回の題材
1990年代のパチンコ「バブル」と、その後の衰退。フィーバー機がもたらした
大量出玉の時代と、過熱の裏で進んだ依存・規制・客離れを、モデル化した
一軒のホール「グランドホール光（ひかり）」（仮名・架空）の盛衰として描く。
実在の店舗・人物・メーカーは登場させず、すべてモデル化している。

## 登場人物
- 語り手:（老人 / 元・グランドホール光の店長 / 仮名「沢田」。今は引退した回想の主）
- 当事者:（若き日の沢田 / 同じ人物の壮年期 / 熱狂を信じ、やがて店を畳む）
- 常連客:（タカハシ / 仮名 / 開店当初からの常連。光の側にも影の側にも立つ一人の客）

## 構成（起承転結）
- 起: 現在。シャッターの下りたビルの前に立つ老人。記憶が巻き戻る。
- 承: バブル期。行列、フィーバーの連鎖、雇用と活気。街が店を中心に回った「光」。
- 転: 過熱の影。のめり込む客、空く家庭の椅子。やがて時代と制度が変わり、客が引く。
- 結: 閉店。老人の問い。熱狂の正体を断じず、余韻として読者に手渡す。

## 監修メモ
- 脚色した点:
  - 店「グランドホール光」、語り手「沢田」、常連「タカハシ」はいずれも創作・モデル化した架空の存在。実在の店舗・個人・メーカーは登場させていない。
  - P9 の家庭の食卓・電話の場面は、依存が生活に及ぼす影響を読者に伝えるための再構成（具体的事実ではない）。
  - 出玉箱・行列・グラフなどの「規模」は誇張せず、印象に留めた。
- 要・事実確認:
  - フィーバー機／CR機（カードシステム）の登場・普及の具体的な時期。
  - 出玉規制・のめり込み（依存）対策など、規制強化の制度名と時期。
  - 1990年代の市場規模・店舗数・推移の数値（作中では具体数を出していないが、改稿で触れる場合は要裏取り）。
  - 三店方式・換金の法的整理を本文で説明する場合の正確な記述。
- 表現上の留意点:
  - 必勝法・勝てる台の見分け方など実用的攻略情報は記載していない（鉄則2）。
  - 依存は当事者を見下さず、「のめり込ませる構造の問題」として描いた（鉄則2／知識メモ）。
  - 実在の固有名詞を避け、断定的に特定対象を悪く描かないようにした（鉄則3）。
