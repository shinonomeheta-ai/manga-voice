# 『ななしちゃん@がんばらない』シナリオ制作プロジェクト

パチンコ・パチスロの実在の事件・不正を、フィクションを通じて紹介する
雑学漫画『ななしちゃん@がんばらない』のシナリオ（ネーム指示書）を量産する。
Claude Code はセッション開始時にこのファイルを自動で読み込む。

## まず読むこと

- `ナレッジ.md` … 作品の文脈・確定設定・これまでの決定事項。**制作前に必ず参照**。
- `設定/作品コンセプト.md` … 作品概要・世界観・トーン。
- `設定/キャラクター設定.md` … ななし／あかり／桐原／九重／九頭竜の詳細。
- `テンプレート/シナリオ出力形式.md` … 7パート構成の出力フォーマット。

## フォルダ構成

- `ナレッジ.md` … 引き継ぎ用コンテキスト
- `設定/作品コンセプト.md` … 作品概要・世界観
- `設定/キャラクター設定.md` … 登場人物の詳細
- `設定/パチンコ知識メモ.md` … 事件・不正・用語のリファレンス（育てる前提）
- `資料/` … 各話のネタ元（実在の事件メモ）。1ファイル＝1事件
- `テンプレート/シナリオ出力形式.md` … 7パート構成の統一フォーマット
- `出力/` … 完成シナリオの保存先（`第NN話_タイトル.md`）
- `.claude/skills/scenario/SKILL.md` … 生成手順本体。`/scenario` で起動

## 制作の鉄則（全話共通）

1. 実在の事件は「桐原が元業界紙記者として解説する」形で扱い、
   登場人物・店舗・メーカー名は**架空名に置換**する（命名は実在のもじりで統一）。
2. **不正のやり方そのものを詳細に解説（教則化）しない**。
3. 特定の実在人物・店舗・メーカーの名誉毀損になる描写をしない。反社絡みは慎重に。
4. 業界を一方的に「悪」と描かない。善良な業界人も登場させる。
   プレイヤーを馬鹿にしない。依存症を扱うときはシリアスに。
5. トーンはコメディ＋シリアスのバランス型。1話完結（事件1つ＝1〜数話）。
6. 出力は必ず `テンプレート/シナリオ出力形式.md` の7パート構成に従う。
7. 年号・数字・固有名詞は裏取りし、未確認は監修メモに「要・事実確認」として残す。

## 使い方

    /scenario 資料/（事件メモ）.md      # 既存の事件メモから1話作る
    /scenario 遠隔操作疑惑              # テーマ・事件名を直接渡す

題材を渡すと、設定とキャラを踏まえて7パート構成のシナリオを生成し、
`出力/` に保存する。
