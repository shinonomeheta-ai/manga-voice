# パチンコ・ドキュメンタリー漫画 シナリオ制作プロジェクト

このリポジトリは、パチンコ業界を題材にしたドキュメンタリー漫画の
シナリオ（ネーム指示書）を量産するためのものです。
Claude Code はセッション開始時にこのファイルを自動で読み込みます。

## まず読むこと

- `ナレッジ.md` … プロジェクトの文脈・これまでの決定事項・技術判断の引き継ぎ。
  シナリオを作る前に必ず目を通し、ここの方針に沿って制作する。

## フォルダ構成

- `ナレッジ.md` … 引き継ぎ用コンテキスト（文脈・決定事項のまとめ）
- `設定/作品コンセプト.md` … 作品全体のトーン・ターゲット・連載仕様。**まずここを埋める**
- `設定/パチンコ知識メモ.md` … 用語や事実関係のリファレンス。事実誤認を防ぐための土台
- `資料/` … 各話のネタ元（取材メモ・実話・台データなど）を1ファイル1題材で入れる
- `テンプレート/シナリオ出力形式.md` … 生成するシナリオの統一フォーマット
- `出力/` … 完成したシナリオの保存先
- `.claude/skills/scenario/SKILL.md` … 生成手順本体。`/scenario` で起動

## 制作の鉄則（全シナリオ共通）

1. **事実が主役**。脚色は読者の理解を助ける範囲に留め、創作した部分は
   出力末尾の「監修メモ」に必ず明記する。
2. **ギャンブルを煽らない**。勝ち方の指南や射幸心を直接あおる描写はしない。
   依存・家計・規制など、ドキュメンタリーとしての多面性を保つ。
3. 実在の個人・店舗・メーカーを断定的に悪く描かない。必要なら仮名・モデル化する。
4. 1話完結を基本としつつ、連載として積み上がる構成にする。
5. 出力は必ず `テンプレート/シナリオ出力形式.md` の形式に従う。

## 使い方

新しい話を作るときは Claude Code 上で:

    /scenario 資料/伝説の台_CR花満開.md

題材ファイルを指定するか、テーマ名を直接渡す（例: `/scenario 1990年代の換金問題`）。
