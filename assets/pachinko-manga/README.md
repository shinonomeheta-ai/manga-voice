# 『ななしちゃん@がんばらない』シナリオ量産キット

Claude Code でシナリオ（ネーム指示書）を、確定設定どおり・7パート構成で量産する一式。

## セットアップ

1. このフォルダを丸ごと作業用の場所に置く。
2. ターミナルでこのフォルダに移動して Claude Code を起動:

       cd pachinko-manga
       claude

3. 起動時に `CLAUDE.md` → `ナレッジ.md` が読まれ、作品の文脈が自動で渡る。

## 使い方

### 事件メモから1話作る
`資料/` に事件メモを置いてから:

    /scenario 資料/サンプル_遠隔操作疑惑.md

### 事件・テーマ名を直接渡す

    /scenario 設定詐欺トラブル

完成シナリオは `出力/第NN話_タイトル.md` に7パート構成で保存される。

### 量産（バッチ）
起動後にこう頼むだけ:

> 資料/ にある全ファイルから1話ずつシナリオを作って、出力/ に保存して

## 構成ファイル

- `CLAUDE.md` … 起動時に自動で読まれる「作品の憲法」（鉄則・地図）
- `ナレッジ.md` … 確定設定・決定事項・配慮事項・縦軸の引き継ぎ
- `設定/作品コンセプト.md` … 作品概要・世界観
- `設定/キャラクター設定.md` … ななし／あかり／桐原／九重／九頭竜
- `設定/パチンコ知識メモ.md` … 事件・不正・用語のリファレンス（育てる前提）
- `テンプレート/シナリオ出力形式.md` … 7パート構成の出力フォーマット
- `.claude/skills/scenario/SKILL.md` … `/scenario` の生成手順
- `資料/` … 事件メモ（ネタ元）／`出力/` … 完成シナリオ

## カスタマイズ

- キャラの口調・設定 → `設定/キャラクター設定.md`
- 回の構成（パートの粒度） → `テンプレート/シナリオ出力形式.md`
- 生成手順そのもの → `.claude/skills/scenario/SKILL.md`
- 事実の蓄積（精度向上） → `設定/パチンコ知識メモ.md` に取材結果を足していく
