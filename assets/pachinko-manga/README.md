# パチンコ・ドキュメンタリー漫画 シナリオ量産キット

Claude Code でシナリオ（ネーム指示書）を統一品質で量産するための一式です。

## セットアップ（3分）

1. このフォルダを丸ごと作業用の場所に置く。
2. ターミナルでこのフォルダに移動して Claude Code を起動:

       cd pachinko-manga
       claude

3. 最初に **`設定/作品コンセプト.md`** の［ ］を自分の作品に合わせて埋める。
   （Claude Code に「作品コンセプトを一緒に埋めて」と頼んでもOK）

## 使い方

### 既存の題材メモから作る
`資料/` にネタ元を1ファイル置いてから:

    /scenario 資料/サンプル_釘師という職人.md

### テーマだけ渡して作る

    /scenario 1990年代のパチンコバブルと衰退

完成したシナリオは `出力/第NN話_タイトル.md` に保存されます。

### 量産する（バッチ）
複数まとめて作りたいときは、起動後にこう頼むだけ:

> 資料/ にある全ファイルから1話ずつシナリオを作って、出力/ に保存して

## 仕組みのポイント

- `CLAUDE.md` … 起動時に自動で読まれる「作品の憲法」。鉄則とフォルダ地図。
- `.claude/skills/scenario/SKILL.md` … `/scenario` の手順本体。設定→資料→
  テンプレの順に読んでから書く、という流れを固定しているので品質がブレない。
- `設定/` を充実させるほど精度が上がる。特に `パチンコ知識メモ.md` に
  事実を貯めていくと、回を重ねるごとに賢くなります。

## カスタマイズのコツ

- ページ数や話数構成を変えたい → `設定/作品コンセプト.md`
- 出力の形（コマ指示の粒度など）を変えたい → `テンプレート/シナリオ出力形式.md`
- 生成の手順そのものを変えたい → `.claude/skills/scenario/SKILL.md`
