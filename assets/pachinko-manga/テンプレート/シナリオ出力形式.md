# シナリオ出力形式（この7パート構成に従って生成する）

> 『ななしちゃん@がんばらない』の標準回フォーマット。全話この構造で書く。

---

# 第NN話「（タイトル）」

- 回タイプ: 標準回 / シリアス回 / コメディ回 / 連作回（N話中M話目）/ 依頼パターン回 / 桐原回想回
- 扱う事件・テーマ: （モデルにした実在事件。※店舗・人物・メーカー名は架空に置換）
- ログライン: （この回が何を見せる話かを一文で）
- 登場キャラ: ななし / あかり / 桐原 / 九重 / 九頭竜 / （その他）
- 主な舞台: 桐原調査事務所 / （メインホール＝未定）/ 錦花町 など

## 構成（7パート）

### ① ツカミ（あかり中心）
- ねらい: 日常会話の中であかりが何かを言い出し、読者を引き込む。
- 内容: （あかりが負ける/勝つ/変な噂を聞く 等）
- ネーム:
  - P1 コマ1【画】（描写）／【セリフ/ナレ】「……」
  - P1 コマ2【画】…／【セリフ】「……」

### ② 疑問の発生（ななし）
- ねらい: ななしが違和感・疑問を持ち、話の入口を作る。
- ネーム: （ページ・コマ単位で）

### ③ 相談（ななし＋桐原／事務所）
- ねらい: ななしが事務所で桐原に話を持っていく。日常の絵作りの基点。
- ネーム:

### ④ 解説（桐原ナレーション）
- ねらい: 桐原が元業界紙記者の経験・資料から実在事件を紹介（劇中劇の導入）。
- 注意: 不正の手口を教則化しない。史実と脚色の線引きを意識。
- ネーム:

### ⑤ 事件パート（九重・九頭竜らの再現ドラマ）
- ねらい: 事件の再現。九重が九頭竜を操る/利用し、被害者が生まれる構図。
- 注意: 人物・店舗名は架空。業界を一方的に悪と描かない。
- ネーム:

### ⑥ 現代に戻る（ななし＋桐原）
- ねらい: 解説が終わり、ななしが感想を述べる。事件の意味を読者に着地させる。
- ネーム:

### ⑦ オチ（あかり）
- ねらい: あかりの一言 or 後日談で締める。
  ※シリアス回は軽いオチにせず、余韻で締めてよい。
- ネーム:

## 監修メモ
- モデルにした実在事件: （何の事件か）
- 架空名への置換: （実 → 架空。店舗・人物・メーカー）
- 創作・脚色した点: （事実と創作の線引き）
- 要・事実確認: （年号・数字・固有名詞など裏取りが必要な箇所）
- 配慮した点: （教則化回避／名誉毀損回避／依存症の扱い／反社の扱い 等）
