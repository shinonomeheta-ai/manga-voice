# キャラクター設定資料（キャラバイブル）

ここにキャラの**顔リファレンス画像**と**プロフィール**を置くと、解析(`analyze`/`run`)の
精度が上がります。特に「漫画コマで誰が喋っているか」の話者同定に効きます。

## 置き方

キャラ1人につき、ファイル名の**stem（拡張子を除いた部分）をキャラ名**にして置きます。

```
assets/characters/
├─ 太郎.png      # 顔リファレンス画像（.png/.jpg/.jpeg/.gif/.webp）
├─ 太郎.md       # プロフィール（性格・口調・声質など。.md/.txt）
├─ 花子.png
└─ 花子.md
```

- 画像だけ／プロフィールだけ でも可。
- プロフィールは `config/characters.json` の `description` ともマージされます。
- 画像名は漫画中の話者名と一致させると、顔照合で speaker が安定します。

> このフォルダの `.md` はサンプルです。実際の顔画像（`太郎.png` 等）を追加してください。
